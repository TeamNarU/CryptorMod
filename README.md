# CryptorMod
A Minecraft mod that adds Cryptor (common) and Cryptor X (Boss) and some stuff. 

---------------------------------------------------------------------------

Adds:
- Common Cryptor
- Cryptor X (boss)
- Cryptor X's sword
- Legendary food
- Cryptor X's armor
- ... and other.
